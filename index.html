07 // Compliance Transparency

Regulatory Standards

A dedicated overview of how the platform meets global regulatory requirements — from OFAC sanctions to EU AI Act auditability.

OFAC Sanctions Screening

Every wallet query is screened against the current OFAC SDN list in real time. Matches are flagged with confidence scores and supporting transaction evidence.

FATF Travel Rule

Infrastructure supports FATF Travel Rule compliance for VASPs — originator and beneficiary identification is structurally embedded into data pipelines.

AML Monitoring

Continuous anti-money laundering pattern detection across indexed chains. Layering, structuring, and mixing behaviors flagged within the 12ms latency window.

EU AI Act Alignment

All ML-driven decisions carry immutable audit logs compatible with EU AI Act Article 13 transparency requirements. Exportable in structured JSON or PDF formats.

Immutable Audit Trails

Every entity flag, alert, and data query is time-stamped and cryptographically signed. Court-admissible documentation available on request for law enforcement.

Data Residency

ClickHouse clusters are deployed with configurable data residency for EU, US, and APAC jurisdictions. No cross-border data movement without explicit client consent.
